# Admixtures Buddy Dashboard

## How to Update Your Website
Whenever you make changes to the code, run these 3 commands in your terminal to publish them to the live website:

```bash
git add .
git commit -m "Update"
git push
```

**Wait 1 minute, then check your phone.**

## Shortcut
You can also just run this single command:
```bash
./deploy_app.sh
```
